# Azure DevOps Terraform Project

## 🚀 Overview
This project uses Terraform to provision Azure infrastructure using a modular approach.

## 🧱 Modules Included
- Virtual Machine (VM)
- Network Interface (NIC)
- Network Security Group (NSG)
- Subnet
- Virtual Network
- Load Balancer
- Bastion
- Resource Group

## ⚙️ Features
- Reusable Terraform modules
- Environment-based deployment (dev, qa, prod)
- Scalable infrastructure setup
- Clean folder structure
